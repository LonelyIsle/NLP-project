To run the code examples in this directory:

1. **Install python**:
    Make sure you have Python 3.7 or higher installed on your machine. You can download it from [python.org](https://www.python.org/downloads/).

2. **To get each method to run**:
    ```bash
    python boolean_search.py
    python m1.py
    python m2.py
    ```

3. **Check the outputs**:
    Each script saves its output in the outputs folder with a matching filename.
